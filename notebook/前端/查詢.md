表格是從 axios.get 取得資料, 用 useState 存起來
`const [data,setData] = useState([])`
使用 array.filter, 將陣列資料做篩選 setData
還需要沒傳查詢字串時,顯示原始資料


取得資料後用二個state存放
* 一個 state 存放原始資料
* 一個 state 用來做為表格顯示用

篩選時對**原始資料**做,因為沒有改變到本身資料,只是將結果給 **顯示用 state**
當沒有查詢字串時,再將原始資料給 **顯示用 state**

```javascript
const [originalData, setOriginalData] = useState([])  // 原始資料
const [data, setData] = useState([])                  // 顯示用資料

// 取得資料時，兩個都設定
useEffect(() => {
  axios.get('/api/items').then(res => {
    setOriginalData(res.data)
    setData(res.data)
  })
}, [])

// 篩選函式
const handleSearch = (query) => {
  if (!query) {
    // 沒有查詢字串 → 還原原始資料
    setData(originalData)
  } else {
    const filtered = originalData.filter(item =>
      item.name.toLowerCase().includes(query.toLowerCase())
    )
    setData(filtered)
  }
}

```

放一個文字輸入框, 將內容存在 state, onChange 時,改變 state
按下查詢鈕->將 state 傳給篩選函式

```jsx
const [query,setQuery]=useState("");

const handleInputChange = (e)=> {
	setQuery(e.target.value)
}

<input onChange={handleInputChange} />
<button onClick={()=>handleSearch(query)} />
```

### 查詢數字欄位
將欄位值和查詢值都轉成數字
```javascript

const handleQuery = (query) => {
    if (!query) {
      setData(originalData);
    } else {
      const filtered = originalData.filter(
        (item) => Number(item.od) === Number(query),
      );
      setData(filtered);
    }
  };

```

### 多欄位查詢
依欄位有無傳入值,針對有值的欄位進行 and 查詢
例: od,sn 這些欄位,分別判斷

用`||`判斷,空字串就條件成立,不會執行該項後面篩選
如果2個欄位都傳空字串,2條篩選都不會做,等同傳回原始所有資料
- od 欄位用 `Number` 轉型,去除 `.00` 或 `.50` 
	- `10.00=>10`
	- `20.50=>20.5`
- sn 欄位使用模糊比對

```javascript
 const handleSearch = () => {
    const filtered = originalData.filter((item) => {
      const matchOd = params.od === "" || Number(item.od) === Number(params.od);
      const matchSn = params.sn === "" || item.sn.includes(params.sn.trim());
      return matchOd && matchSn;
    });
    setData(filtered);
 };
