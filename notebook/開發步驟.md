1. 建立資料表
```
CREATE TABLE funds (
    id SERIAL PRIMARY KEY,  -- ID
    in_date DATE,           -- 買入日期
    qty SMALLINT,           -- 股數
    price SMALLINT          -- 單價          
);
```

2. 建立 Express 路由

routes/funds.js
子路由,特定資料表所需操作(CRUD)都寫在此檔
get,post,put,delete

utils/db-helper.js
組合 sql 語句

server.js
將子路由引入,設定路徑使用該檔
