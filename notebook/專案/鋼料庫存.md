wp_stock 欄位
demand_id 關聯至 wp_demand.id
日後可用來查詢那些庫存用在那些需求

```sql
SELECT 
    d.*, 
    -- 即時統計 stock 表中，歸屬於此需求且狀態為成品的數量
    (SELECT COUNT(*) 
     FROM wp_stock s 
     WHERE s.demand_id = d.id AND s.status = 'product') AS completed
FROM wp_demand d;
```

要在原本資料表加入此欄位
```sql
ALTER TABLE wp_stock 
ADD COLUMN demand_id INTEGER REFERENCES wp_demand(id);
```

1. 建好 pg 資料表 (create 指令放在 sql 資料夾)


2. 使用 echoway/backend/migration/migrate.js
做匯入(設定 access 資料庫路徑
`D:\Dev\echoway2026\stocks.mdb`)

3. tablesConfig.js
設定來源資料表欄位和目的地資料表欄位

4. 命令列輸入 `node migrate.js`
