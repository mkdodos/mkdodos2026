用 puppeteer 套件將 html 直接列印 pdf , 
寫 node.js 腳本在命令列即可運行

```javascript
import puppeteer from 'puppeteer';

(async () => {
  // 啟動瀏覽器
  const browser = await puppeteer.launch();
  const page = await browser.newPage();
  
  // 前往目標網頁
  await page.goto('https://example.com');
  
  // 拍照存檔
  await page.screenshot({ path: 'example.png' });

  // 關閉瀏覽器
  await browser.close();
  console.log('截圖完成！');
})();
```

node screenshot.js

如果想做一個「輸入網址就幫忙轉成 PDF」的線上工具：

1. 用 **Express** 架設伺服器，開一個 `POST /generate-pdf` 的 API 路由。    
2. 當使用者呼叫這個 API 時，Express 在後端觸發 **Puppeteer** 打開瀏覽器、導向該網址、將網頁導出成 PDF。    
3. Express 再將 PDF 檔案回傳給使用者。
