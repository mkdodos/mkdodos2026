axios 傳資料到 express 取得的方式

```javascript
// axios get params
const runCut = async (record) => {  
    const url = `${API_BASE}/stock-fit-cut`;
    const { demand_id, stock_id } = record;
    const response = await axios.get(url, { params: { demand_id, stock_id } });
  };

// express req.query
router.get("/stock-fit-cut", async (req, res) => {
  res.send(req.query.demand_id);  
});
```


>`axios.get` 帶 `params` 會自動轉成 query string
>/stock-fit-cut?demand_id=xxx&stock_id=yyy

express 取得前端傳來參數的方式
```
req.query.id      // GET 參數：/stock-fit-cut?id=123
req.params.id     // 路由參數：/stock-fit-cut/:id
req.body.id       // POST body
```
