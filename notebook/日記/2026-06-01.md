node 專案開機自動執行

- 前端 vite react
- 後端 express
- web server nginx

### Vs Code
開發時,使用 vs code , tasks
建立 .vscode 資料夾 , 建立 tasks.json

```json
{

  "version": "2.0.0",
  "tasks": [
    {
      "label": "自動啟動前端1 (Frontend)",
      "type": "npm",
      "script": "dev",
      "path": "my-project/frontend", // 告訴 VS Code package.json 的相對位置
      "options": {
        "cwd": "${workspaceFolder}/my-project/frontend" // 實際執行指令的子目錄
      },
      "presentation": {
        "reveal": "silent",
        "panel": "new"
      },
      "runOptions": {
        "runOn": "folderOpen" // 開啟大資料夾時自動執行
      }
    },
    {
      "label": "自動啟動前端2 (Frontend)",
      ...  
    },
  ]
}
```

### Express
使用 [[PM2]] 套件, 在開機時執行 node 檔案
pm2  -> express -> node server.js

### 前端建置
建置 build 之後, 用 [[Nginx]] 伺服器
winsw 將 nginx 註冊成 win service , 就可在開機時自動執行
winsw-> nginx -> build 

nginx conf 設定 server ip 和 port (對應 express)
