原資料表 Stocks 有數量欄位,將其依數量拆分為多筆
方便日後切割後可針對單支溯源
處理方式
在 pg 建立 old_stock 資料表
使用 migrate.js 將 Stocks 資料匯入
使用 insert into select 語法將 old_stock 資料新增至 wp_stock 


* 目前資料庫為 sql server,將它匯出成 access

* 將 access 匯入至 pg

1. 建好 pg 資料表 (create 指令放在 sql 資料夾)


2. 使用 echoway/backend/migration/migrate.js
做匯入(設定 access 資料庫路徑
`D:\Dev\echoway2026\stocks.mdb`)

3. tablesConfig.js
設定來源資料表欄位和目的地資料表欄位

4. 命令列輸入 `node migrate.js`
